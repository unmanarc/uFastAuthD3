Length Requirements
Minimum number of characters required.
Maximum number of characters allowed.
Character Type Requirements
Characters considered as "special". Password must contain characters from this set.
Pattern & Complexity Rules
Max same characters in a row (0 = no limit). E.g., 3 blocks "aaaa" but allows "aaa".
Minimum number of distinct characters the password must contain (0 = no limit).
Max percentage of the password that can be the same character (1 - 100).
Block sequential patterns (e.g., "abc", "123", "xyz"). Max Count sets the maximum length of sequential pattern to detect (0 = no limit).
Security Rules
Reject passwords from a list of commonly used / breached passwords.
Password cannot contain the user's username (case-insensitive).
Non-Compliance Action
Action to take when password does not meet strength requirements during password introduction at login.
